OrderControl

Sistema TPV para restaurantes llamado OrderControl, desarrollado con Angular en el frontend y Spring Boot en el backend

Interfaz Táctil

En próximas versiones, optimizado para tablets y pantallas táctiles

Gestión Completa

Control de mesas, productos y categorías

Multi-Mesa

Gestión simultánea de múltiples mesas

Facturación

Sistema completo de cobro e impresión

Tecnologías Utilizadas

A

Frontend - Angular

Interfaz de usuario moderna y responsive desarrollada con Angular, con vistas a ser optimizada para dispositivos táctiles y experiencia de usuario fluida.

SB

Backend - Spring Boot

API REST robusta y escalable construida con Spring Boot, proporcionando toda la lógica de negocio y gestión de datos.
